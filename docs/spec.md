# Aorazer 現行確定仕様

## 1. 基本コンセプト

* 名前は **Aorazer**
* 読みは **アオレイザー**
* モチーフは **天を仰ぐ / 双眼 / 上向きのまなざし**
* 左右対称性と、身体性を損なわない入力体験を重視する

---

## 2. 初号機の方針

* **有線 split** 前提
* **無線は後回し**
* **中央拡張モジュール構想はいったん捨てる**
* 中央は

  * **磁石による左右連結**
  * 必要なら **受動アタッチメント（Alice角など）**
    までに留める
* 中央の能動拡張は将来課題

---

## 3. 構成

* 左右分割キーボード
* **片側1MCUずつ**
* 左右は **TRRS** で接続
* 左右連結用に **磁石** を使う
* トップ/ボトムプレートやケースは初号機では必須ではなく、まずは動作優先で進める

---

## 4. MCU / ファーム方針

* ファームは **QMK**
* **VIA対応**
* **Mod Tap対応**
* MCUは **RP2040系本命**
* 無線化するなら将来別リビジョンで検討
* `nRF52840 + ZMK` は将来の無線版候補だが、初号機では採らない

---

## 5. キー数 / 行列

### 片側

* 通常キー数: **39**
* ジョイスティック: **1**
* エンコーダー: **1**

### 行列

* **5行8列でOK**

---

## 6. 操作子

### ジョイスティック

* 左右それぞれに **1個ずつ**
* **Nintendo Switch互換品** を採用候補
* **押し込みあり**
* 方向入力と押し込みは QMK/VIA 管理前提

### エンコーダー

* 左右それぞれに **1個ずつ**
* **人差し指で回す位置** に再配置
* 押し込みは**不要寄り**で、初号機では前提にしない方向

---

## 7. LED

* RGBバックライトあり
* LEDは **SK6812 MINI-E**
* HaLIK用の余剰分を使う前提

---

## 8. レイアウト更新点

* `Fn` キーと `Delete` キーを入れ替え済み
* 最新のJSONレイアウトが現行版


---

## 9. Emacsキー仕様

### 配置

* 左中段の `Ctrl` 位置を **Emacs** キーに変更

### 役割

* **OS差を吸収する主修飾キー**
* `A/E/H/K/B/F/P/N` については Emacs風編集動作
* それ以外は

  * **Windowsでは Ctrl**
  * **Macでは Command**
    として振る舞う

### 特別挙動

* `Emacs + A` → 行頭へ
* `Emacs + E` → 行末へ
* `Emacs + H` → Backspace
* `Emacs + K` → カーソル以降を削除
* `Emacs + B` → Left
* `Emacs + F` → Right
* `Emacs + P` → Up
* `Emacs + N` → Down

### 補足

* `Ctrl+F` など既存ショートカット競合は、**左下の通常Ctrlキー**で解決する前提

---

## 10. OS判定

* **QMKの OS Detection を使う前提**
* `Emacs` キーの

  * Win時 Ctrl
  * Mac時 Command
    の切替に使う
